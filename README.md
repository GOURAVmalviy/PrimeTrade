# Bitcoin Market Sentiment vs Trader Performance Analysis

## Objective

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using Hyperliquid historical trading data.

## Dataset

- Historical Trader Data
- Bitcoin Fear & Greed Index

## Project Workflow

1. Data Cleaning
2. Feature Engineering
3. Data Merging
4. Exploratory Data Analysis
5. Statistical Analysis
6. Business Insights
7. Trading Strategy Recommendations

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Scikit-learn

## Key Insights

- Average profitability varied across different market sentiment categories.
- Trading behavior changed under Fear and Greed conditions.
- Trading sessions and selected coins influenced profitability.
